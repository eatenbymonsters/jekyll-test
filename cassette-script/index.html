Remember the old C90 tapes?

Here's my visual homage to the ubiquitous music format of my youth. This is "Cassette Script"; a handwriting typeface with over 200 characters.

Cassette Script Beta desktop version

Cassette Script Beta webfonts

Use the links above to download the beta versions of Cassette Script. The desktop version is in True Type format (.ttf), and the webfont version contains .ttf .eot .svg and .woff file types along with a copy of the associated css rules (see code box below): everything you need to use Cassette Script in your own web pages.


    @font-face {
      font-family: 'cassette';
      src: url('../cassetteWebfont/cassette00.eot');
      src: url('../cassetteWebfont/cassette00.eot?#iefix') format('embedded-opentype'),
           url('../cassetteWebfont/cassette00.woff') format('woff'),
           url('../cassetteWebfont/cassette00.ttf') format('truetype'),
           url('../cassetteWebfont/cassette00.svg#cassette') format('svg');
      font-weight: normal;
      font-style: normal;
    }
